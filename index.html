Privacy Policy

This privacy policy explains how YouTube Ad Blocker works, what data it handles, and how it uses browser permissions. The extension is designed to be privacy-first and operate only on YouTube pages.

Product: YouTube Ad Blocker Version: 1.0.0 Effective Date: 2026-06-24
Summary: YouTube Ad Blocker does not collect, store, sell, or transfer personal data. It works locally in your browser and uses only the minimum permissions needed to function on YouTube.

Information We Collect

We do not collect personal information, financial information, health information, location data, browsing history, or content from your web activity.

The extension does not send your data to any external server.

Local Data Stored on Your Device

  • Your enable/disable preference for the extension
  • A local counter showing how many ads were blocked

This information is stored locally using chrome.storage.local and remains on your device.

How the Extension Works

YouTube Ad Blocker operates only on YouTube pages. It may hide ad-related elements, detect supported ad placements, and apply browser-side blocking rules to reduce interruptions while you watch videos.

The extension does not function on unrelated websites and does not read or monitor your browsing activity across the web.

Permissions We Use

  • declarativeNetRequest — Used to apply static browser-side rules that help block ad-related requests on YouTube.
  • scripting — Used to run extension scripts on YouTube pages so the extension can detect and manage supported page elements.
  • tabs — Used to identify active YouTube tabs so the extension can operate only where needed.
  • storage — Used to save your local settings and the blocked-ad counter.
  • *.youtube.com/* host access — Used only to run the extension on YouTube pages.

Data Sharing

We do not sell, rent, or share user data with third parties. The extension does not include advertising SDKs, analytics tools, or tracking services.

Remote Code

The extension does not use remote code. All scripts, styles, and assets are packaged inside the extension and are included in the version submitted to the Chrome Web Store.

Children’s Privacy

The extension is not intended to collect information from children and does not knowingly collect personal data from any user.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated effective date.
